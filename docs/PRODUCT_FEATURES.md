# AgentGuard 产品功能清单

**面向用户的功能文档** - 您能用 AgentGuard 做什么

---

## 📦 当前版本: v0.3.0

### ✅ 核心功能

#### 1. AI Agent 安全扫描

**功能**: 自动检测和评估系统中运行的 AI Agents

```bash
agentguard scan
```

**您将看到**:
- ✅ 检测到的 Agent 列表 (Claude Code, Cursor IDE, OpenClaw 等)
- ✅ 每个 Agent 的运行状态 (运行中/已停止)
- ✅ 安全评分 (0-100 分)
- ✅ 发现的安全问题
- ✅ 修复建议

**使用场景**:
- 每天早上检查 Agent 状态
- 发现 Agent 异常时排查问题
- 定期安全审计

---

#### 2. Token 使用统计

**功能**: 追踪和分析 AI 服务的 Token 消耗和成本

```bash
agentguard tokens
```

**您将看到**:
- ✅ 今日/本周/本月成本汇总
- ✅ 各 Agent 的 Token 使用明细
- ✅ 输入/输出 Token 分布
- ✅ 成本估算 (基于官方定价)

**真实数据示例**:
```
💰 成本汇总:
  今日:   $147.01
  本周:   $312.05
  本月:   $1,792.69

🤖 Claude Code:
  本月: 32.2M tokens → $1,449.19

🤖 Cursor IDE:
  本月: 15.6M tokens → $343.50
```

**使用场景**:
- 每周查看成本趋势
- 月底对账和预算管理
- 优化 AI 使用习惯

---

#### 3. 报告生成

**功能**: 生成美观的 HTML 安全报告

```bash
agentguard export --format html
```

**报告内容**:
- ✅ 安全评分可视化
- ✅ Agent 状态总览
- ✅ 详细问题列表
- ✅ Token 成本统计
- ✅ 修复建议

**报告特点**:
- 📱 响应式设计 (手机/平板/电脑)
- 🎨 美观的紫色渐变主题
- 🖨️ 打印友好格式
- 📧 方便分享 (单个 HTML 文件)

**使用场景**:
- 团队安全审计报告
- 向管理层汇报成本
- 个人存档和对比

- 🔔 **预算告警** - 成本超标实时提醒

**使用场景**:

- 团队协作 (多人同时访问)
- 实时告警 (成本异常立即知道)

---

#### 2. 预算管理

**功能**: 设置成本预算，避免超支

**设置预算**:
```bash
# 通过配置文件设置
~/.agentguard/config.json

{
  "budget": {
    "daily": 50,      // 日预算 $50
    "weekly": 300,    // 周预算 $300
    "monthly": 1000   // 月预算 $1000
  }
}
```

**告警机制**:
- 🟢 **70% 使用** → 提醒 "预算已用 70%"
- 🟡 **90% 使用** → 警告 "预算即将用尽"
- 🔴 **100% 使用** → 严重告警 "预算已用尽"

**通知方式**:
- 💻 Dashboard 横幅提醒
- 🔔 系统桌面通知
- 📧 邮件通知 (可选)

**使用场景**:
- 个人成本控制
- 团队预算管理
- 防止意外超支

---

#### 3. 历史趋势分析

**功能**: 查看和对比历史数据

**对比维度**:
- 📅 **今日 vs 昨日** - 今天花了更多还是更少?
- 📅 **本周 vs 上周** - 本周使用是否正常?
- 📅 **本月 vs 上月** - 月度成本趋势如何?
- 📅 **自定义时间** - 任意时间段对比

**可视化**:
- 📈 折线图: 每日成本趋势
- 📊 柱状图: 各 Agent 成本占比
- 🥧 饼图: Token 使用分布

**使用场景**:
- 发现使用规律
- 优化成本结构
- 月度/季度汇报

---

## 🎯 使用流程

### 日常监控流程

```bash
# 1. 启动 Dashboard
agentguard serve

# 2. 浏览器自动打开 http://localhost:3000
# 3. 查看实时数据
#    - 当前运行的 Agent
#    - 今日 Token 消耗
#    - 成本趋势图表

# 4. 收到告警时处理
```

### 周报生成流程

```bash
# 1. 通过 Dashboard 查看本周数据
# 2. 对比上周数据
# 3. 导出 PDF 报告
# 4. 发送给团队/上级
```

### 成本优化流程

```bash
# 1. 查看历史趋势图

#    - 减少不必要的 Token 消耗
#    - 选择更经济的 Agent
#    - 设置合理的预算限制
```

## 💡 最佳实践

### 1. 每日检查

**建议**: 每天早上打开 Dashboard 查看

**关注点**:
- ✅ 所有 Agent 是否正常运行
- ✅ 昨日成本是否异常
- ✅ 是否有安全问题
- ✅ 预算使用情况

**耗时**: 2-3 分钟

---

### 2. 预算设置

**建议**: 根据实际情况设置合理预算

**参考标准**:
- 💵 **个人开发者**: $50-100/月
- 👥 **小团队 (3-5人)**: $300-500/月
- 🏢 **中型团队 (10-20人)**: $1000-2000/月

**调整策略**:
- 第一个月观察实际消耗
- 第二个月设置预算 (实际消耗 × 1.2)
- 每月根据需求调整

---

### 3. 成本优化

**优化方向**:

1. **减少不必要的使用**
   - 避免重复生成相同内容
   - 使用缓存结果
   - 精简 Prompt

2. **选择合适的 Agent**
   - 简单任务用 Cursor (更便宜)
   - 复杂任务用 Claude (更强大)
   - 根据任务分配

3. **设置使用限制**
   - 日预算限制
   - 单次使用上限
   - 自动暂停功能

---

### 4. 团队协作

**适用场景**: 多人共同使用 AI Agent

**协作方式**:
- 📊 **共享 Dashboard** - 团队成员都能访问
- 📧 **告警通知** - 成本异常及时知道
- 📄 **周报分享** - 定期汇报使用情况

**管理建议**:
- 设置团队总预算
- 分配各成员配额
- 定期回顾和优化
