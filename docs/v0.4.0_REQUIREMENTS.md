# AgentGuard v0.4.0 需求规划

**规划日期**: 2026-03-08
**目标版本**: v0.4.0 (Feature Release)
**预计发布**: 2026-03-22 (2 周开发周期)
**核心主题**: **Web Dashboard + 预算管理 + 智能告警**

---

## 🎯 版本定位

### v0.3.0 → v0.4.0 升级路径

| 版本 | 定位 | 核心功能 | 用户群 |
|-----|------|---------|-------|
| v0.3.0 | CLI 工具 | 扫描、Token 统计、报告 | 开发者 (命令行用户) |
| **v0.4.0** | **CLI + Web** | **+ Dashboard + 预算告警** | **开发者 + 团队** |
| v1.0.0 | 企业级平台 | + 多用户 + 实时监控 | 企业团队 |

### 战略转向: Electron → Web Dashboard

**决策依据**:
1. ❌ Electron 环境问题难以解决 (耗时 > 8 小时未解决)
2. ✅ Web Dashboard 开发更快 (React + Express, 预计 3 天)
3. ✅ 更好的跨平台支持 (浏览器即可访问)
4. ✅ 易于部署和分享 (团队协作场景)
5. ✅ 更低的维护成本 (无需打包分发)

**技术栈选择**:
- 前端: React 18 + TypeScript + Tailwind CSS + Recharts
- 后端: Express + TypeScript + WebSocket (实时更新)
- 数据: SQLite (Token 缓存) + JSON (配置)

## 📊 用户功能对比 (v0.3.0 → v0.4.0)

当前用户使用流程:
```bash

# 2. 打开 HTML 文件查看
```

**限制**:
- ❌ 数据不会自动更新 (需要重新生成)
- ❌ 无法交互 (只能看，不能点)
- ❌ 无法导出其他格式 (只有 HTML)
- ❌ 无历史数据对比
- ❌ 无成本告警

新的用户体验:

**用户价值提升**:

```json
    "daily": 50,      // 日预算 $50
    "weekly": 300,    // 周预算 $300
    "monthly": 1000,  // 月预算 $1000
    "alerts": {
      "thresholds": [0.7, 0.9, 1.0],  // 70%, 90%, 100%
      "channels": ["desktop", "email"] // 通知渠道
    }
  },
  "locale": "zh-CN",  // 语言设置
  "theme": "light"    // 主题
}
```

**告警逻辑**:

```typescript
class BudgetMonitor {
  async checkBudget(period: 'daily' | 'weekly' | 'monthly'): Promise<Alert | null> {
    const usage = await this.tokenService.getUsage(period);
    const budget = this.config.budget[period];
    const percentage = usage / budget;

    if (percentage >= 1.0) {
      return { level: 'critical', message: `${period} 预算已用尽` };
    } else if (percentage >= 0.9) {
      return { level: 'warning', message: `${period} 预算已用 90%` };
    } else if (percentage >= 0.7) {
      return { level: 'info', message: `${period} 预算已用 70%` };
    }

    return null;
  }
}
```

**告警渠道**:
1. **Dashboard 横幅** - 红色/黄色告警条
2. **系统通知** - macOS Notification Center
3. **邮件通知** (可选) - SMTP 配置

#### 5. Token 缓存启用

**集成到 TokenTracker**:

```typescript
// src/core/token-tracker.ts
import { TokenCache } from './token-cache';

export class TokenTracker {
  private cache: TokenCache;

  constructor() {
    this.cache = new TokenCache();
  }

  async getUsageReport(period: Period): Promise<TokenReport> {
    await this.cache.init();

    // 1. 尝试从缓存读取
    const cached = await this.cache.getCachedUsage(
      agentId,
      startDate,
      endDate
    );

    if (cached.length > 0) {
      return this.buildReport(cached);
    }

    // 2. 计算实际数据
    const usage = await this.calculateUsage(period);

    // 3. 写入缓存
    await this.cache.cacheUsage(agentId, new Date(), usage);

    return this.buildReport(usage);
  }
}
```

**缓存策略**:
- 每日自动缓存 (凌晨 2:00)
- 手动触发: `agentguard tokens --cache`
- 保留 90 天数据
- 自动清理过期数据
