## v0.2.0 - 扩展 Agent 支持（2-3周）

### 核心目标
> **让 AgentGuard 支持 3+ 主流 AI Agents**

### 功能开发

#### 1. Claude Desktop 完整支持 🎯
**优先级**: 🔴 最高

**检测能力**：
- [ ] 自动发现 Claude Desktop 进程
- [ ] 解析 `~/Library/Application Support/Claude/` 配置
- [ ] MCP servers 配置读取

**安全检查**（6项）：
- [ ] MCP server 权限审计（严重）
  - 检查 filesystem、database、exec 等危险权限
  - 评估 MCP server 来源可信度
- [ ] Computer Use 模式检查（严重）
  - 是否启用屏幕控制
  - 鼠标键盘权限
- [ ] 配置文件权限（中危）
  - config.json 是否 600 权限
- [ ] 自动启动项检查（低危）
- [ ] 网络连接审计（中危）
- [ ] 数据目录权限（中危）

**自动修复**（3项）：
- [ ] 禁用危险 MCP servers
- [ ] 修正配置文件权限
- [ ] 限制 Computer Use 权限

#### 2. Cursor 完整支持 🎯
**优先级**: 🟠 高

**检测能力**：
- [ ] 发现 Cursor 进程
- [ ] 解析 `~/.cursor/` 配置

**安全检查**（5项）：
- [ ] Workspace Trust 检查（严重）
- [ ] 扩展来源验证（高危）
- [ ] AI 索引范围审计（中危）
- [ ] 端口绑定检查（中危）
- [ ] 远程连接审计（低危）

**自动修复**（2项）：
- [ ] 启用 Workspace Trust
- [ ] 限制索引敏感目录

#### 3. 实时监控功能 📊
**优先级**: 🟡 中

- [ ] 进程监控（CPU/内存使用）
- [ ] 网络连接监控
- [ ] 配置文件变更监控
- [ ] TUI 界面（使用 blessed 或 ink）

#### 4. 测试完善
- [ ] Claude Desktop 单元测试（覆盖率 >80%）
- [ ] Cursor 单元测试（覆盖率 >80%）
- [ ] 集成测试套件

## v0.3.0 - 企业级功能（1-2个月）

### 核心目标
> **让 AgentGuard 成为企业可用的合规工具**

### 功能开发

#### 1. Token 使用统计 💰
**优先级**: 🔴 最高（企业需求）

**功能**：
- [ ] 解析各 Agent 的日志文件
  - OpenClaw: `~/.openclaw/logs/`
  - Claude Desktop: `~/Library/Logs/Claude/`
- [ ] 提取 Token 用量数据
  - 输入 Token 数
  - 输出 Token 数
  - API 调用次数
- [ ] 成本计算
  - 支持多模型定价（Opus/Sonnet/Haiku）
  - 日/周/月报表
- [ ] 异常检测
  - Token 用量突增告警
  - 疑似滥用检测

**CLI 命令**：
```bash
agentguard tokens                    # 查看所有 Agent Token 用量
agentguard tokens openclaw --today   # 查看今日用量
agentguard tokens --cost             # 显示成本
agentguard tokens --alert            # 用量告警
```

#### 2. 合规报告导出 📄
**优先级**: 🟠 高

**格式支持**：
- [ ] PDF 报告（使用 puppeteer）
  - 公司 Logo 定制
  - 执行摘要
  - 详细检查项
  - 修复建议
- [ ] HTML 报告
  - 可交互式
  - 图表展示
- [ ] Excel 报告
  - 原始数据导出
  - 多 Agent 对比

**模板**：
- [ ] ISO 27001 合规模板
- [ ] SOC 2 审计模板
- [ ] 内部安全审计模板

#### 3. GitHub Copilot 支持 🎯
**优先级**: 🟡 中

**检查项**（4项）：
- [ ] 遥测设置审计
- [ ] 代码建议来源检查
- [ ] 网络代理配置
- [ ] 敏感文件过滤

#### 4. 豆包桌面版支持 🎯
**优先级**: 🟢 低

**检查项**（4项）：
- [ ] 权限请求审计（相机/麦克风）
- [ ] 网络连接审计
- [ ] 自动更新机制检查
- [ ] 隐私设置审计

## 优先级说明

### 🔴 最高优先级
- Claude Desktop 支持（v0.2.0）
- Token 使用统计（v0.3.0）

**原因**：
- Claude Desktop 用户量大，需求强烈
- Token 成本是企业核心关注点

### 🟠 高优先级
- Cursor 支持（v0.2.0）
- 报告导出（v0.3.0）

**原因**：
- 开发者常用工具
- 合规审计必需

### 🟡 中优先级
- 实时监控（v0.2.0）
- GitHub Copilot 支持（v0.3.0）
- 沙箱检测（v0.4.0）

### 🟢 低优先级
- 豆包支持（v0.3.0）
- GUI 应用（v1.0.0）
- 云服务版（v1.0.0）
