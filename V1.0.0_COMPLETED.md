- **React 18.2.0** - UI 框架

## ✨ 核心功能

### 1. Dashboard (主界面)

#### 统计概览
- 🎯 安全评分（带动画仪表盘）
- 🤖 活跃 Agent 数量
- 🔴 严重问题统计
- 💰 今日成本显示

#### Agent 管理
- 📊 每个 Agent 的详细安全评分
- 🔍 问题列表（带颜色编码）
- 🔧 一键自动修复
- ⚙️ 停止/重启控制

#### Token 统计
- 💵 日/周/月成本统计
- 📈 预算追踪（进度条可视化）
- ⚠️ 预算告警系统
- 📑 每个 Agent 详细分解

### 2. Floating Widget (浮窗)

- 📏 320x200 紧凑设计
- 🔝 Always-on-top（始终置顶）
- 🖱️ 可拖动标题栏
- 🔄 实时数据更新（每 10 秒）
- 📊 关键指标一览
  - 总体安全评分
  - Agent 数量
  - 严重问题数
  - 今日成本

### 3. System Tray (系统托盘)

- 🎨 托盘图标（支持自定义）
- 📋 右键菜单
  - Show Dashboard
  - Toggle Floating Widget
  - Scan Now
  - Quit
- 👆 左键点击显示主窗口
- 🔄 后台运行（关闭窗口不退出）

### 4. Auto Scan (自动扫描)

- ⏰ 每 10 秒自动扫描
- 📡 结果推送到所有窗口
- 🔄 数据实时同步
- 🔔 严重问题通知（预留接口）

## 🐛 已修复问题

### 构建阶段

1. ✅ **TypeScript 配置错误**
   - 问题: `allowImportingTsExtensions` 冲突
   - 解决: 调整 tsconfig.main.json 配置

2. ✅ **未使用变量警告**
   - 问题: 多个未使用的参数和变量
   - 解决: 添加 `_` 前缀或禁用检查

3. ✅ **App.isQuitting 属性不存在**
   - 问题: TypeScript 找不到该属性
   - 解决: 创建全局 `isQuitting` 变量

4. ✅ **图标路径问题**
   - 问题: 托盘图标文件不存在
   - 解决: 添加 fallback 逻辑

   - 问题: Tailwind CSS 无法编译
   - 解决: 创建 postcss.config.js
