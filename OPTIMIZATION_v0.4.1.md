```typescript
// src/desktop/main/index.ts
function snapToEdge(x, y, width, height) {
  const { width: screenWidth, height: screenHeight } = screen.getPrimaryDisplay().workAreaSize;
  const SNAP_DISTANCE = 30;

  // 边界限制逻辑
  if (x < SNAP_DISTANCE) newX = 10;
  if (x + width > screenWidth - SNAP_DISTANCE) newX = screenWidth - width - 10;
  // ...
}
```

## 🚀 使用指南

### 启动应用
```bash
cd agentguard
npm run desktop
```

### 体验新功能

1. **拖拽悬浮球**
   - 拖动到屏幕边缘 → 自动磁吸
   - 释放后位置自动保存
   - 重启应用自动恢复位置

2. **悬停查看**
   - 鼠标悬停悬浮球500ms
   - 显示今日消耗和安全评分
   - 快速预览无需点击

3. **点击打开面板**
   - 单击悬浮球 → 打开简洁模式
   - 显示关键指标 + Agent状态 + 快速操作
   - 高度360px，信息精简

4. **展开详细信息**
   - 点击"查看 Token 详情"
   - 面板高度变为600px
   - 显示Token分解和问题列表

5. **快速操作**
   - 点击"Dashboard" → 打开Web UI
   - 点击"扫描" → 立即扫描所有Agents
   - 无需展开即可使用
