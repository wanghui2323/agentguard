⚠️ 安装说明

AgentGuard 目前还未发布到 npm 仓库，请使用以下方式安装：

方法 1: 从 GitHub 克隆（推荐）

git clone https://github.com/wanghui2323/agentguard.git cd agentguard npm install npx tsx src/cli/index.ts scan

方法 2: 直接使用 CLI

cd /Users/wanghui/Desktop/AI写作空间/agentguard npm install npx tsx src/cli/index.ts --help

✨ 核心功能

🔍 自动发现

无需配置，自动检测系统中运行的所有 AI Agents（OpenClaw、Claude Desktop、Cursor等）

🛡️ 安全扫描

全面检查端口配置、身份认证、权限设置、版本漏洞等8大安全项

📊 风险评分

0-100分量化评分系统，清晰展示安全状态（优秀/良好/警告/危险）

🔧 一键修复

自动修复6类常见问题，自动备份配置，修复失败可回滚

🎛️ 进程控制

统一管理所有 Agent 的启动、停止、重启操作

📄 报告导出

生成 JSON/PDF 格式的安全报告，支持合规审计

🚀 快速开始

  1. 克隆项目
    git clone https://github.com/wanghui2323/agentguard.git
  2. 安装依赖
    cd agentguard npm install
  3. 扫描 AI Agents
    npx tsx src/cli/index.ts scan
  4. 修复安全问题
    npx tsx src/cli/index.ts fix openclaw
